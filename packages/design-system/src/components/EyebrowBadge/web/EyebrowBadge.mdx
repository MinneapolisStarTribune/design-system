# EyebrowBadge

Eyebrow Badges highlight top-priority newsroom content and signal urgency or editorial significance. They appear above headlines-just like standard Eyebrow Labels-but are reserved for the highest-visibility stories across the site, including homepage leads, section fronts, and article pages.

Badges come in Showcase, Breaking, Live, and Sponsored variants, each representing a distinct editorial state. They may include a secondary label for timestamps or series titles. By default the root is a **`span`**; use **`as="a"`** or **`as={NextLink}`** with **`href`** when the badge should navigate (the link is the root—no wrapper, with **`cursor: pointer`** on hover).

## Example: Breaking badge with Next.js Link

In your app, use **`NextLink`** from **`next/link`**. Storybook renders a mock with the same **`href`** contract.

```tsx
import NextLink from 'next/link';
import { EyebrowBadge } from '@minneapolisstartribune/design-system/web';

<EyebrowBadge
  as={NextLink}
  href="/breaking"
  prefetch={false}
  label="Breaking"
  secondaryLabel="Developing story"
  variant="breaking"
  size="large"
/>
```
