# Social Embeds

Social Embeds provide a layout wrapper for third party social content such as Instagram Facebook Twitter and Threads.  
The component ensures consistent spacing alignment and responsive width behavior within article layouts.

## Key concept

This component follows a slot pattern

The Design System handles layout and responsiveness  
The consumer provides the embed content  

## When to use

Embedding social media posts inside article content  
Maintaining consistent widths across platforms  
Ensuring responsive behavior across breakpoints  
Keeping embed logic separate from layout  

## Anatomy

Container  
Outer wrapper that controls width spacing and alignment  

Embed slot  
Children passed by the consumer which renders the embed content

## Usage

```tsx
<SocialEmbeds
  platform="instagram"
  aria-label="Instagram post"
>
  <InstagramEmbed url="..." />
</SocialEmbeds>
```

## Behavior

Width is controlled using semantic tokens  
Layout adapts across desktop tablet and mobile  
Content remains centered and responsive  
No platform specific embed logic is handled internally  

## Out of scope

Rendering third party embeds  
Managing platform SDKs  
Handling data fetching or scripts
