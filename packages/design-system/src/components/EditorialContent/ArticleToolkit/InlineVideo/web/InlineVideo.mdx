# InlineVideo

The **InlineVideo** component renders video content within editorial layouts, supporting captions and video credits.
It provides a responsive container for embedding video players while maintaining consistent layout across article templates.

[Figma Comps](https://www.figma.com/design/f8DSKwKjG4MsSqmA59v1um/Core-Components?node-id=2449-81728)

## Variants

### Standard

Used within standard article layouts with constrained width.

### Immersive

Used within immersive layouts where video can span wider content areas.

## Orientation

- **Horizontal** (`horizontal`) — Default landscape layout.
- **Vertical** (`vertical`) — Centered portrait layout with constrained width.

## Usage guidelines

### When to use

- Embed video within editorial content
- Provide context using captions and credits

### When to avoid

- Avoid using as a standalone video player
- Avoid for multi-video grid layouts

## Props

- **variant** – `'standard' | 'immersive'`
- **orientation** – `'horizontal' | 'vertical'`
- **caption** – Optional video description
- **videoCredit** – Optional attribution
- **posterUrl** – Optional thumbnail (used by consumer)
- **children** – Video player element
- **dataTestId** – Optional test id

## Web

Import the web component from `@minneapolisstartribune/design-system/web`.

```tsx
import { InlineVideo } from '@minneapolisstartribune/design-system/web';

<InlineVideo
  variant="standard"
  orientation="horizontal"
  caption="General Manager Heather Ann Mady with Moua outside the building."
  videoCredit="Rebecca McApline/Star Tribune"
  posterUrl="https://images.unsplash.com/photo-1490750967868-88aa4486c946?auto=format&fit=crop&w=1200&q=8"
>
  <video
    src="https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4"
    poster="https://images.unsplash.com/photo-1490750967868-88aa4486c946?auto=format&fit=crop&w=1200&q=8"
    controls
  />
</InlineVideo>;
```

## React Native

Import from `@minneapolisstartribune/design-system/native` and pass the native video player as `children`.
The design system provides the container, caption, credit, and accessibility mapping; the consuming app owns the player implementation.

```tsx
import { InlineVideo } from '@minneapolisstartribune/design-system/native';
import Video from 'react-native-video';

const videoUrl = 'https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4';
const posterUrl =
  'https://images.unsplash.com/photo-1490750967868-88aa4486c946?auto=format&fit=crop&w=1200&q=8';

export function ArticleScreen() {
  return (
      <InlineVideo
        variant="immersive"
        orientation="horizontal"
        caption="General Manager Heather Ann Mady with Moua outside the building."
        videoCredit="Rebecca McApline/Star Tribune"
        posterUrl={posterUrl}
        dataTestId="article-inline-video"
        aria-label="Video: General Manager interview"
      >
        <Video source={{ uri: videoUrl }} poster={posterUrl} controls resizeMode="contain" />
      </InlineVideo>
  );
}
```

`react-native-video` is shown as an example. It is not required by the design system.
