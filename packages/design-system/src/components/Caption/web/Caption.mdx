# Caption

Captions provide context for images and videos within articles. They clarify what the audience is seeing, add essential detail, and ensure proper attribution. This component is informational and non-interactive, and may include navigation controls when paired with galleries or lightboxes.

This component serves as the foundational caption primitive for editorial media components including InlineImage, ImageGallery, PhotoLayout, and Lightbox integrations.

[Figma Comps](https://www.figma.com/design/f8DSKwKjG4MsSqmA59v1um/Core-Components?node-id=3305-1555)

## Anatomy

1. **Caption Text** — Descriptive editorial text that provides additional image context.
2. **Photo Credit** — Attribution for the image source or photographer. Attribution always follows the caption and is enclosed in parentheses.
3. **Photo Purchase Link (optional)** — Optional CTA allowing users to purchase a reprint of the image.
4. **Gallery Controls (optional)** — Previous/next navigation when paired with a carousel or gallery.
5. **Counter (optional)** — Current position indicator (e.g. `1/17`) for lightbox layouts.

## Variants

### Inline

Used within editorial article body layouts on light surfaces.

### Lightbox

Used within immersive or overlay-based experiences on dark surfaces.

## Configurations

### Purchase Link

Use when readers should be able to purchase a reprint of the image.

### Gallery navigation

Pass `currentIndex`, `totalItems`, and `onPrevious` / `onNext` when the caption is paired with a gallery. Inline layouts render controls in the top row; lightbox layouts render the counter and controls in a bottom row.

## Usage guidelines

### When to use

- Display editorial image captions and credits consistently.
- Standardize attribution formatting across media experiences.
- Provide optional photo purchase functionality.
- Support future gallery and lightbox integrations through shared APIs.

### When to avoid

- Avoid for generic form field helper text or non-editorial captions.
- Avoid using as a standalone layout container for media experiences.

## Accessibility

- Uses semantic `<figcaption>` structure.
- Purchase CTA remains keyboard accessible.
- Lightbox variant maintains accessible contrast ratios on dark backgrounds.

## Props

- **caption** – Optional editorial caption content.
- **credit** – Optional attribution or photographer credit.
- **purchaseLink** – Optional CTA configuration object `{ label, link, onClick }`.
	- CTA renders only when `link` is present and non-empty.
	- If `label` is missing or blank, it defaults to **Buy Reprint**.
- **variant** – Caption presentation variant: `'inline' | 'lightbox'`.
- **currentIndex** / **totalItems** – Optional gallery pagination for controls and counter.
- **onPrevious** / **onNext** – Optional gallery navigation handlers.
- **analytics** – Optional metadata merged into the Buy Reprint tracking event.
- **onPurchaseLinkClick** / **onNavigationClick** – Optional interaction callbacks.
- **className** – Optional custom class names.
- **dataTestId** – Optional data-testid attribute for testing.
