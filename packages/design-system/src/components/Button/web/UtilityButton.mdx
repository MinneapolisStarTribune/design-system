# Utility Button

Utility Buttons are compact, lightweight controls designed for dense UI surfaces such as navigation bars, article headers, toolbars, and cards. They support two content styles—icon-only and icon + label—to offer clarity across different breakpoints and layouts. Utility Buttons surface quick, low-hierarchy actions like share, print, search, or close.

[Figma Comps](https://www.figma.com/design/f8DSKwKjG4MsSqmA59v1um/Core-Components?node-id=358-40809&m=dev)

## Usage guidelines

### Content styles
- **Icon-only:** best for space-constrained actions where the icon is immediately recognizable.
- **Icon + label:** best when recognition needs extra clarity across breakpoints or in denser UI contexts.

### Where to use
- Navigation bars, article headers, toolbars, and cards.

### Common actions
- Share, print, search, and close.
