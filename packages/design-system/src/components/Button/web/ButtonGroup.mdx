# Button Group

`ButtonGroup` lays out multiple `Button` components in a horizontal row with consistent spacing.
When there is not enough horizontal space, buttons wrap to the next line while keeping a fixed vertical row gap.

## Usage guidelines

- Use `ButtonGroup` when related actions should be presented together.
- `ButtonGroup` handles layout and spacing only.
- Each child `Button` owns its own `variant`, `color`, `size`, and behavior.
- For icon-only buttons, pass an explicit `aria-label` on each button.

## Layout behavior

- Orientation is horizontal only (`flex-direction: row`).
- Buttons can wrap to additional rows (`flex-wrap: wrap`).
- Wrapped rows use `--spacing-8` vertical spacing.

## Usage example

```tsx
import { Button, ButtonGroup } from '@minneapolisstartribune/design-system/web';
import { AvatarIcon } from '@minneapolisstartribune/design-system/web/icons';

export function Example() {
  return (
    <ButtonGroup>
      <Button variant="filled" color="brand" size="small" onClick={() => {}}>
        Save
      </Button>
      <Button variant="outlined" color="neutral" size="small" onClick={() => {}}>
        Cancel
      </Button>
      <Button
        variant="outlined"
        color="neutral"
        size="small"
        icon={<AvatarIcon />}
        aria-label="Open profile options"
        onClick={() => {}}
      />
    </ButtonGroup>
  );
}
```
