# SwiperCarousel

**SwiperCarousel** is a flexible, reusable carousel component built on top of Swiper.

It provides core carousel behavior (scrolling, pagination, navigation) while allowing full control over layout composition.

## Usage guidelines

- Use for horizontal scrollable content
- Compose layouts using:
  - SwiperCarousel.Slide
  - SwiperCarousel.Pagination
  - SwiperCarousel.Navigation
- Order of children controls layout

## Slides Behavior

### slidesPerView

- 'auto' (default): content-based width, multiple visible slides, scroll-based navigation
- 1: single slide view, ideal for image gallery
- 2,3,...: fixed slides per view, pagination per group

## Pagination behavior

- Disabled by default  
- Enable using SwiperCarousel.Pagination  
- Uses Swiper native pagination  
- Styled via design system tokens  
- Automatically hides when not scrollable  

## Navigation behavior

- Optional  
- Enable using SwiperCarousel.Navigation  
- Automatically disables at boundaries  
- Works with slidesPerView="auto"  
- Uses real scroll position  
- Supports custom button props  

## Layout flexibility

### Image gallery
- slidesPerView={1}
- Caption left
- Navigation right
- Pagination inside

### Cards layout
- slidesPerView="auto"
- Centered pagination + navigation

### Top controls
- Title left
- Navigation right
