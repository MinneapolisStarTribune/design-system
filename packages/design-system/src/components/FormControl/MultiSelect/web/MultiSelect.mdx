# MultiSelect

MultiSelect inputs allow users to choose multiple values from a predefined list of options. They are useful in forms and filtering flows where selecting more than one value is expected. MultiSelect supports multiple sizes, rounded styling, disabled/error/success states, and `FormGroup` integration for accessible labels and helper messaging.

`<MultiSelect>` keeps the dropdown open while users toggle selections, and displays chosen labels in the trigger as comma-separated text.

`MultiSelect` works standalone, but when wrapped in `FormGroup`, it automatically connects labels and captions via `aria` attributes.

## Basic usage

```tsx
import { FormControl, FormGroup } from '@minneapolisstartribune/design-system/web';

<FormGroup>
  <FormGroup.Label>Options</FormGroup.Label>
  <FormControl.MultiSelect
    id="topics"
    options={[
      { value: 'news', label: 'News' },
      { value: 'sports', label: 'Sports' },
      { value: 'culture', label: 'Culture' },
    ]}
  />
</FormGroup>;
```

## Controlled usage

```tsx
import { useState } from 'react';
import { FormControl } from '@minneapolisstartribune/design-system/web';

const [value, setValue] = useState<string[]>([]);

<FormControl.MultiSelect id="topics" value={value} onChange={setValue} options={OPTIONS} />;
```

```tsx

```

## States

```tsx
<FormControl.MultiSelect options={OPTIONS} />
<FormControl.MultiSelect value={['option-1', 'option-2']} options={OPTIONS} />

```

## Error state

```tsx
<FormGroup>
  <FormGroup.Label>Options</FormGroup.Label>
  <FormControl.MultiSelect error options={OPTIONS} />
  <FormGroup.Caption variant="error">Please select at least one option</FormGroup.Caption>
</FormGroup>
```

## Accessibility

- role="combobox" with aria-expanded
- aria-controls links combobox to listbox
- role="listbox" with aria-multiselectable="true"
- role="option" with aria-selected on each option
- aria-labelledby via FormGroup
- fallback aria-label when no label is provided

## Integration with FormGroup

```tsx
<FormGroup>
  <FormGroup.Label>Options</FormGroup.Label>
  <FormGroup.Description>Select one or more topics</FormGroup.Description>

  <FormControl.MultiSelect options={OPTIONS} />

  <FormGroup.Caption variant="info">Choose one or more options</FormGroup.Caption>
</FormGroup>
```

## Truncation behavior

Selected labels are shown in the trigger as comma-separated text and use single-line truncation with ellipsis when content exceeds available width.

In Storybook, the configurable example uses a constrained width so truncation is easy to verify.
