# Select

`<Select>` allows users to choose a single option from a dropdown list. It supports keyboard navigation, multiple sizes, states, and integrates with `FormGroup` for accessibility.

## Architecture
Layer	What it does	Required?
FormGroup + FormControl	Layout, labels, descriptions, accessibility	Recommended
Select	Dropdown behavior and selection	Always


`Select` works standalone, but when wrapped in `FormGroup`, it automatically connects labels and descriptions via `aria` attributes.

## Basic usage

```tsx
<FormGroup>
  <FormGroup.Label>Country</FormGroup.Label>
  <FormControl.Select
    id="country"
    options={[
      { value: 'us', label: 'United States' },
      { value: 'ca', label: 'Canada' },
    ]}
  />
</FormGroup>

Controlled usage
const [value, setValue] = useState<string | undefined>();

<FormControl.Select
  id="country"
  value={value}
  onChange={setValue}
  options={OPTIONS}
/>;

Sizes

medium (default)
large
<FormControl.Select size="small" />
<FormControl.Select size="medium" />
<FormControl.Select size="large" />

pointy (default)

<FormControl.Select radius="rounded" />

<FormControl.Select placeholderText="Select an option" />

Disable placeholder:

<FormControl.Select showPlaceholder={false} />
States
Default
<FormControl.Select options={OPTIONS} />
Filled
<FormControl.Select value="us" options={OPTIONS} />
Error
<FormGroup>
  <FormGroup.Label>Country</FormGroup.Label>
  <FormControl.Select isError options={OPTIONS} />
  <FormGroup.Caption variant="error">
    This field is required
  </FormGroup.Caption>
</FormGroup>
Disabled
<FormControl.Select isDisabled options={OPTIONS} />
Accessibility
role="combobox" with aria-expanded
aria-controls links combobox to listbox
aria-activedescendant for keyboard navigation
aria-labelledby via FormGroup
fallback aria-label when no label is provided
Keyboard interaction
| Key         | Behavior              |
| ----------- | --------------------- |
| ArrowDown   | Open / move down      |
| ArrowUp     | Move up               |
| Enter       | Select option         |
| Escape      | Close dropdown        |
Integration with FormGroup
<FormGroup>
  <FormGroup.Label>Country</FormGroup.Label>
  <FormGroup.Description>
    Choose your country
  </FormGroup.Description>

  <FormControl.Select options={OPTIONS} />

  <FormGroup.Caption variant="info">
    Informational message
  </FormGroup.Caption>
</FormGroup>
