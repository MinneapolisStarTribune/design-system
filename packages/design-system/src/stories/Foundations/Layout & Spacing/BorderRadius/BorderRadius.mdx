# Border Radius

Border radius tokens define consistent corner radius values used throughout the design system. These tokens are available as CSS custom properties.

## Usage

Border radius tokens can be used in several ways:

### CSS Custom Properties

```css
.my-element {
  border-radius: var(--radius-8);
  border-top-left-radius: var(--radius-16);
}
```
