<h2 style={{ marginTop: '2rem', marginBottom: '0.75rem' }}>Using button colors in React and CSS</h2>

**`Button`** uses `color="neutral" | "brand" | "brand-accent"`—module **variants**, not semantic tokens like `brand-01` (see [**Classes vs. token keys**](#classes-vs-token-keys)). [Button component docs](?path=/docs/actions-button--docs). **Using these tokens in code** has the category table, SCSS / inline / theme, and the same **Classes vs. token keys** section.

```tsx
import { Button } from '@minneapolisstartribune/design-system/web';

<Button color="brand" variant="filled" size="medium">
  Subscribe
</Button>
```

**Defaults:** `color="neutral"`, `variant="filled"`. **Type:** `ButtonColor` (`Button.types.ts`).
