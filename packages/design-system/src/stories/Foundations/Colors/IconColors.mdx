<h2 style={{ marginTop: '2rem', marginBottom: '0.75rem' }}>Using icon colors in React and CSS</h2>

Barrel icons: `color` → `var(--color-icon-…)` on the SVG (`currentColor` on paths), same token **keys** as text—not arbitrary hex. [**Classes vs. token keys**](#classes-vs-token-keys) below covers `color` prop vs CSS vars for text/icon; **Using these tokens in code** has the full table, typography note, Button contrast, SCSS, and theme.

```tsx
import { CameraIcon } from '@minneapolisstartribune/design-system/web';

<CameraIcon size="small" color="on-dark-secondary" />
```

**Defaults:** `size="medium"`, `color="on-light-primary"` (`Icon.tsx`). **Type:** `IconColor` (`Icon.types.ts`). Barrel `*Icon` components use the same short token keys as text (`brand-01`, `on-light-primary`, …), mapped to `var(--color-icon-*)`. **`Button`** uses a separate `color` API (`neutral` / `brand` / `brand-accent`), not these keys. **Icons gallery & controls:** [Icons > Docs](?path=/docs/foundations-icons--docs).
