# Spacing

Spacing tokens define consistent spacing values used throughout the design system. These tokens are available as CSS custom properties and can be used with Tailwind CSS utilities.

## Usage

Spacing tokens can be used in several ways:

### CSS Custom Properties

```css
.my-element {
  padding: var(--spacing-16);
  margin: var(--spacing-24);
  gap: var(--spacing-8);
}
```

### Tailwind CSS Utilities

Tailwind v4 automatically generates utilities from spacing tokens. Simply use the numeric value - Tailwind will automatically use the corresponding CSS custom property:

```jsx
<div className="ds:p-16 ds:m-24 ds:gap-8">
  {/* Automatically uses var(--spacing-16), var(--spacing-24), var(--spacing-8) */}
</div>
```

**How it works:** When you write `ds:p-16`, Tailwind automatically maps it to `var(--spacing-16)`. This happens automatically - no need to write `p-[var(--spacing-16)]` or reference the CSS variable directly.

**Note:** This project uses the `ds:` prefix for all Tailwind utilities.

## Spacing Scale

The spacing scale ranges from `0` to `96px`, providing a comprehensive set of spacing values for all use cases.
