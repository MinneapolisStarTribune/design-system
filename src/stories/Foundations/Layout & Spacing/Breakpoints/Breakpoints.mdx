# Breakpoints

Breakpoints define the responsive breakpoints for the design system. Resize your browser window to see how content adapts at different breakpoints.

## CSS Custom Properties

```css
@media (min-width: var(--breakpoint-large-min)) {
  /* Large breakpoint styles */
}

@media (min-width: var(--breakpoint-medium-min)) and (max-width: var(--breakpoint-medium-max)) {
  /* Medium breakpoint styles */
}

@media (max-width: var(--breakpoint-small-max)) {
  /* Small breakpoint styles */
}
```
