# Border Radius

Border radius tokens define consistent corner radius values used throughout the design system. These tokens are available as CSS custom properties and can be used with Tailwind CSS utilities.

## Usage

Border radius tokens can be used in several ways:

### CSS Custom Properties

```css
.my-element {
  border-radius: var(--radius-8);
  border-top-left-radius: var(--radius-16);
}
```

### Tailwind CSS Utilities

Tailwind v4 automatically generates utilities from border radius tokens. Simply use the numeric value - Tailwind will automatically use the corresponding CSS custom property:

```jsx
<div className="ds:rounded-8 ds:rounded-tl-16">
  {/* Automatically uses var(--radius-8), var(--radius-16) */}
</div>
```

**How it works:** When you write `ds:rounded-8`, Tailwind automatically maps it to `var(--radius-8)`. This happens automatically - no need to write `rounded-[var(--radius-8)]` or reference the CSS variable directly.

**Note:** This project uses the `ds:` prefix for all Tailwind utilities.
